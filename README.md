## Title
A feature-rich to-do list web application built using HTML, CSS, JavaScript, jQuery, and Bootstrap. The app allows users to manage tasks efficiently with visual priority segregation and a theme toggle for better user experience.

This project focuses on combining JavaScript logic with UI frameworks to build an interactive and user-friendly front-end application.

## Features
- Add, update, and delete tasks
- Priority-based task color segregation
- Light and dark theme toggler
- Responsive layout using Bootstrap
- Interactive UI powered by jQuery

## Tech Stack
- HTML
- CSS
- JavaScript
- jQuery
- Bootstrap

## Project Structure

index.html – application layout and structure

style.css – custom styling and theme colors

script.js – task logic, priority handling, and theme toggling

## What I Learned
- Using jQuery for DOM manipulation and event handling
- Integrating Bootstrap for responsive UI design
- Implementing theme toggling functionality
- Managing UI state and priority-based styling
- Structuring a front-end project with multiple libraries
